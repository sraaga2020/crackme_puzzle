# crackme_puzzle

## Description
A suspicious message was recovered from a compromised system.

The message has been transformed multiple times to hide its contents.

## Objective
Recover the hidden message.

## Hints

1. The first layer uses a common computer representation system.
2. ASCII may help translate the first layer.
3. The final layer uses a simple historical cipher.
4. Look at how characters change rather than trying random guesses.

## Instructions

1. Click the green **Code** button on this GitHub repository.
2. Select **Download ZIP**.
3. Extract the ZIP file.
4. Open a terminal in the extracted repository folder.
5. Run this command to start playing the puzzle: ./crackme
